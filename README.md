# Risk Calculator

## Overview
This project contains an Excel-based Risk Calculator designed to assess and calculate risk based on user-provided inputs.

## Features
- User-friendly interface
- Automatic calculations using Excel formulas
- Quick risk assessment
- Easy to customize

## Files
- `Risk Calculator.xlsx` – Main Excel workbook
- `screenshots/` – Preview images of the workbook

## Tools Used
- Microsoft Excel

## How to Use
1. Download the workbook.
2. Open it in Microsoft Excel.
3. Enter the required input values.
4. Review the calculated risk results.
